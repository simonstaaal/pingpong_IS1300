# Github setup

[Github cheat-sheet](https://pages.github.com/)
1. **Navigera till mapp**
```
cd /sökväg/till/din/mapp
```
2. **Initiera ett Git-repo i mappen**
```   
git init
```
4. **Lägg till alla filer**
```
Git add .
```
6. **Gör första commit**
```
git commit -m "Första commit"
```
8. **Koppla mappen till Github Repo**
```
git remote add origin https://github.com/ditt-
användarnamn/ditt-repo.git
```
10. **Ladda upp till github**
```
git push -u origin main
```
